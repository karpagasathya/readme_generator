# README Generator

## Description
This is a README Generator is a command-line application that runs with Node.js which dynamically generates a good README file by answering a few prompts about your project.

## Table of Contents
* [Installation](#installation)

* [Usage](#usage)

* [License](#license)


## Installation

To generate your own README, first run `npm install` in order to install the following npm package dependencies such as:

* [inquirer](https://www.npmjs.com/package/inquirer) that will prompt you for your inputs from the command line.
* [axios](https://www.npmjs.com/package/axios) to fetch your info from the GitHub API
